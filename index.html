Research Experience & Publications:

  • Correlation Power Analysis Attack on AES Using Switching Glitch Power Leakage Model, Jul 2015
    • Technical Report
    • Abstract:Power leakage from hardware implementation of encryption algorithms is a start point for power analysis attacks. Correlation Power Analysis attack is one of power analysis methods that uses correlation between estimated power consumption and real power measurement. This paper uses Correlation Power Analysis attack on a hardware implementation of AES to find embedded secret key. Two power leakage models called Hamming Distance and Switching Glitch is compared for power prediction. It is proven that Switching Glitch model is more efficient than Hamming Distance model in power prediction.
